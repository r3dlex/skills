# Edit Article Skill

## Quick Start

1. Divide article into sections based on headings
2. Confirm sections with user
3. Rewrite each section — max 240 chars/paragraph
4. Preserve core message while tightening prose

## Workflow

### Step 1 — Divide into Sections

Divide based on headings. Consider information dependencies (DAG) — later sections should build on earlier ones.

**Confirm sections with user** before proceeding.

### Step 2 — Rewrite Each Section

For each section:
1. Rewrite for clarity, coherence, and flow
2. Max **240 characters per paragraph**
3. Preserve core message

## Key Principles

- **Information dependencies**: Order sections to respect information flow — later sections should build on earlier ones
- **Tight prose**: Every word must earn its place
- **240 char limit**: Keeps paragraphs scannable and punchy
