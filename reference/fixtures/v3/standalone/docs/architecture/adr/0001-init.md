# ADR 0001: Init

## Status
Accepted.

## Context
The v3 AI-SDLC scaffold replaces the legacy `.agents`-based layout with a canonical `.ai/`, `.memory/`, and `docs/` tree.

## Decision
Adopt the v3 layout described in `modules/documentation-blueprint.md` and `modules/topology.md`.

## Consequences
The legacy `.agents` directory is marked deprecated by the migration rules in `modules/migration.md`.
